# Icons

export const icons = () => {
  // Use vite's ?react loader to import svgs as React components
  const modules = import.meta.glob('./*.svg?react', { eager: true })
  
  const icons = Object.entries(modules).map(([path, { default: Icon }], i) => (
    <div key={i} style={{display: "flex", justifyContent:"space-between", padding: "5px 20px"}}>
      {path.replace(/^\.\/(.+)\.svg\?react$/, '$1')}
      <Icon key={i} height={24} width={24} />
    </div>
  ))

  return icons
}

Icons are loaded using [SVGR](https://react-svgr.com/docs/webpack/) and should all be stored in `src/components/icons`.
Using SVGR allows us to import the svgs directly to avoid cumulative layout shift and style them with ease.
Below are all the icons used in the application. Most of them are from [Boxicons](https://boxicons.com/) and in 24x24.
